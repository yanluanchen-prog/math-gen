# TEST_CHECKLIST.md（发布前冒烟测试清单）

> 规则：每次改动后必跑；10 分钟内跑完；任何一项失败都不发布、不覆盖母版。

## A. 打开与基本 UI
- [ ] 页面能正常打开（无白屏）
- [ ] 控制台无致命报错（红色 error）
- [ ] 手机端：点击按钮有响应（无整页卡死）

## B. 核心流程（按你的游戏改写）
- [ ] 首页 → 开始 → 进入地图/关卡选择
- [ ] 进入关卡 → 出题正常
- [ ] 数字输入/删除/提交正常
- [ ] 正确/错误反馈正常
- [ ] 通关结算页显示正常
- [ ] 返回地图/首页正常

## C. 高风险功能（最容易被改坏）
- [ ] “不会做/提示”弹窗：打开/关闭/动画/步骤正常
- [ ] 错题本：能记录/能展示/能练习
- [ ] 家长面板：能进入（验证）/tab 切换/数据展示
- [ ] 存档：刷新页面后数据仍在（localStorage 不损坏）
- [ ] 语音/音效：开关正常（注意浏览器自动播放限制）

## D. 兼容性快速检查
- [ ] iPhone/Safari（或 iPad）：滚动/点击不穿帮
- [ ] Android/Chrome：滚动/点击正常
- [ ] PC/Chrome：键盘输入（如果支持）正常

## E. 发布前动作
- [ ] 生成锁定版文件（带时间戳）
- [ ] 计算 SHA256 并记录到 VERSIONS.md
- [ ] 更新变更说明（写“改了什么/为什么改/风险点”）
